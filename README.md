# Health Symptom Checker (Advanced)

A professional, rule-based Python & Streamlit project to check symptoms, get possible causes, advice, and risk level.

## Features

- Dropdown selection of symptoms
- Age & gender input
- Severity slider
- Risk calculation
- Professional Streamlit dashboard
